## 🚀 Execution
    1. Clone repository
    2. cd Transfer_Room/frontend_project
    3. npm install
    4. npm run serve
    5. cd Transfer_Room/tranfer/sqls
    6. Database 
        - transfer_ddl.sql
        - transfer_dml_dongCode.dml
        - transfer_dml.sql 순으로 실행
    7. Transfer_Room/transfer/src/main/java/com/transfer/room/TransferApplication.java 실행
